Instructions

  1. Cliquez avec le bouton droit sur chaque image
  2. Sélectionnez "Enregistrer l'image sous..."
  3. Nommez le fichier selon l'indication sous l'image
  4. Enregistrez dans le dossier assets/img/
  5. Remplacez ces images par vos vraies photos quand vous en aurez

Note : Ces images sont générées aléatoirement par picsum.photos et ne représentent pas votre établissement. Elles servent uniquement de placeholder temporaire.
