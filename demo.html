🎓 Site Web École - Démonstration

📋 Instructions de démarrage

1. Ouvrez le fichier index.html dans votre navigateur

2. Téléchargez les images ci-dessous et placez-les dans le dossier assets/img/

3. Personnalisez le contenu selon vos besoins

4. Déployez sur GitHub Pages, Netlify ou votre hébergeur

📁 Structure du projet

Site Casel/ ├── index.html # Page d'accueil ├── presentation.html # Présentation ├── equipe.html # Équipe pédagogique ├── galerie.html # Galerie d'événements ├── blog.html # Blog/Actualités ├── contact.html # Contact ├── README.md # Documentation └── assets/ ├── css/ │ ├── style.css # Styles personnalisés │ └── placeholders.css # Images de démo ├── js/ │ └── script.js # JavaScript └── img/ # Vos images ici

📝 Personnalisation

Facile à modifier :

  • Nom de l'établissement : Recherchez "Keur Serigne Touba" dans tous les fichiers
  • Couleurs : Modifiez les couleurs dans la configuration TailwindCSS
  • Contenu : Éditez directement les fichiers HTML
  • Images : Remplacez les fichiers dans assets/img/
